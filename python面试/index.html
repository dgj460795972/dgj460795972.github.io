python面试，实际就是对python知识掌握能力的一种鉴定。通过面试，可以更好的掌握自己学到的知识。

导入模块

  1. import module 导入模块
  2. from module import member 从模块中导入某成员
  3. from module import * 从模块中导入所有的成员
  4. import module as 别名 增加别名的方式

设置python模块的搜索路径

  1. 设置PYTHONPATN变量 PYTHONPATH = ‘path’
  2. 通过sys.path临时设置python模块包的路径，sys.path.append(‘path’)
  3. 在python安装路径site-packages下增加一个后缀为.pth文件，增加为模块包的路径
